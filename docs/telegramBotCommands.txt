get_image_meta - Get full image info from the gallery by id or slug
random_image - Get a random image from the gallery by title

gallery_admin - Get started with the gallery
upload_image - Upload image to gallery
edit_image - Edit image in gallery
edit_artist - Edit artist in gallery

ei - same as edit_image
ea - same as edit_artist
