# Magic Card Rules

## Set Rules

Colors:
- Black (B)
- Blue (U)
- Green (G)
- Red (R)
- White (W)

Lands:
- Swamp (B)
- Island (U)
- Forest (G)
- Mountain (R)
- Plains (W)

Artifacts:
- 3 artifacts, 1-3 mana

Every color should have an equal number of cards.

- 4 daumens, 1-4 mana (non legendary)
- 1 legendary daumen, 5 mana
- 1 instant
- 1 sorcery
- 1 enchant daumen
- 1 enchantment
- total mana cost of each color set should be 22-25m.
